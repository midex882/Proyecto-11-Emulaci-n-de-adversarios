# Terraform

Esta carpeta contiene el despliegue equivalente al `docker-compose.yml` usando el provider Docker.

## Uso

```bash
cd terraform
terraform init
terraform apply
```

Para destruir:

```bash
terraform destroy
```

En la defensa del proyecto se puede usar Docker Compose para ejecutar la demo y conservar Terraform como automatizacion de infraestructura exigida por la entrega.
